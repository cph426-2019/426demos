Common DOM Uses

May 18th, 2019

The purpose of this page is to give a demo of some common DOM traversal techniques.

  • nodeName gives the name of the Node type's name
  • childNodes gives access to all children of any DOM Node
  • children is a property of the ParentNode interface. It returns an HTMLCollection of all childNodes which are Elements.
  • matches is a method of Element that allows you to test an element against a CSS selector.
